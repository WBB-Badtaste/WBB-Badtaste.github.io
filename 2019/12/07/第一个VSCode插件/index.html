第一个VSCode插件

Posted on 2019-12-07 | Post modified: 2019-12-07 |

介绍

在使用VScode的开发过程中，终会遇到一些简单的开发辅助功能又找不到合适插件的情况。例如：我在编写Markdown的过程中大量需要粘贴截图到文章，并且文章需要多端渲染。因此准备做一个自动上传剪切板数据到OOS并生成URL的插件。

使用模板

  1. 切换到脚手架项目路径，执行npm install -g yo generator-code
  2. yo code初始化项目，这里可能会因网络问题延迟几十秒
